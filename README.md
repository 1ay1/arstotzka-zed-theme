# Arstotzka

> Ultra-vibrant dark theme port for Zed Editor with **EXTREME** glass effects

A port of the Arstotzka theme from [August Themes](https://github.com/inci-august/august-themes) with dramatically enhanced transparency/blur effects, ultra-vibrant colors, and advanced Zed features.

## Installation

1. Clone this repo
2. Copy `themes/arstotzka.json` to:
   - **macOS/Linux**: `~/.config/zed/themes/`
   - **Windows**: `%APPDATA%\Zed\themes\`
3. Restart Zed
4. Select theme: `Cmd/Ctrl+K` ‚Üí `Cmd/Ctrl+T`

## Quick Settings

```json
{
  "theme": {
    "mode": "dark",
    "dark": "Arstotzka (Frosted)"
  },
  "indent_guides": {
    "enabled": true,
    "coloring": "indent_aware"
  }
}
```
